# ImageAnalysisProject2021
Image Analysis Project 2021


## Requirements

Python version used: 3.9
Package manager: anaconda

External libraries:
```bash
tkinter
tkinter.ttk
```

```bash
pip install easyocr
```

```bash
pip install opencv-python
```

```bash
pip install imutils
```

## How to run program
After install all external libraries run command below

```bash
python GUI.py
```
