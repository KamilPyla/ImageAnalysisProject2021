# ImageAnalysisProject2021
Image Analysis Project 2021


## Requirements

Python version used: 3.9
Package manager: anaconda

External libraries:
```bash
opencv-python
tkinter
tkinter.ttk
pip install easyocr
pip install opencv-python
pip install imutils
```

## How to run proram
After install all external libraries run command below

```bash
python GUI.py
```
